## PART 1 — How to Add Your Own Product Photos

### Step 1 — Prepare your images

- Take photos in **good natural light** (near a window)
- Use a **white or plain background** — place product on a white sheet
- Recommended size: **800 × 800 px** minimum, square format works best
- Save as **.jpg** or **.webp** (smaller file size, faster loading)
- Name files clearly: `tape-clear.jpg`, `bubble-wrap-large.jpg`, etc.

---

### Step 2 — Create an `images` folder

Inside your website folder (where your HTML files are), create a folder named `images`:

```
sjcorp/
  ├── index.html
  ├── products.html
  ├── product.html
  ├── cart.html
  └── images/
       ├── tape-clear.jpg
       ├── tape-brown.jpg
       ├── bubble-large.jpg
       ├── bubble-small.jpg
       ├── stretch-500.jpg
       ├── stretch-300.jpg
       ├── carton-3ply.jpg
       ├── carton-5ply.jpg
       ├── carton-7ply.jpg
       └── logo.png
```

---

### Step 3 — Add images to product.html

Open `product.html` in any text editor (Notepad, VS Code, etc.)

Find the `DB` object near the bottom. Each product has an `img:''` field. Add your image path there:

**Before:**
```js
'tape-48x65-clear': {
  name: 'BOPP Adhesive Tape 48mm × 65m (Clear)',
  img: '',
  ...
}
```

**After:**
```js
'tape-48x65-clear': {
  name: 'BOPP Adhesive Tape 48mm × 65m (Clear)',
  img: 'images/tape-clear.jpg',
  ...
}
```

Repeat for every product. The image will automatically show on the product detail page.

---

### Step 4 — Add images to products.html and index.html

In both files, find the product data arrays (look for `icon: '📦'`). Add an `img` field next to each icon:

```js
{ id: 'tape-48x65-clear', name: '...', img: 'images/tape-clear.jpg', icon: '📦', ... }
```

Then find where the product card HTML is built (look for `p-thumb`). Change:

```html
<div class="p-thumb">${p.icon}</div>
```

To:

```html
<div class="p-thumb">
  ${p.img
    ? `<img src="${p.img}" alt="${p.name}" style="width:100%;height:100%;object-fit:cover">`
    : p.icon
  }
</div>
```

---

### Step 5 — Add your company logo

Replace the `SJ` text logo box with your actual logo image. In all 4 HTML files, find:

```html
<div class="logo-box">SJ</div>
```

Replace with:

```html
<img src="images/logo.png" alt="SJ Corporation" style="height:36px;border-radius:9px">
```

#### Approach 2 — Node.js Backend (Full backend support)

Install **Node.js** from https://nodejs.org

Create a file `server.js` in your project:

```js
const express = require('express');
const nodemailer = require('nodemailer');
const cors = require('cors');

const app = express();
app.use(express.json());
app.use(cors());
app.use(express.static('.')); // serve your HTML files

// Email transporter — use Gmail
const transporter = nodemailer.createTransport({
  service: 'gmail',
  auth: {
    user: 'jayendra.jsingh@gmail.com',
    pass: 'YOUR_APP_PASSWORD'   // Gmail App Password (not your login password)
  }
});

// Order endpoint — called when customer places order
app.post('/api/order', async (req, res) => {
  const { name, phone, email, address, items, total } = req.body;

  const itemList = items.map(i =>
    `• ${i.name} × ${i.qty} = ₹${(i.price * i.qty).toLocaleString('en-IN')}`
  ).join('\n');

  const mailOptions = {
    from: 'jayendra.jsingh@gmail.com',
    to: 'jayendra.jsingh@gmail.com',
    subject: `New Order from ${name} — ₹${total}`,
    text: `
NEW ORDER — SJ Corporation
===========================
Customer: ${name}
Phone: ${phone}
Email: ${email || 'Not provided'}
Address: ${address}

ORDER ITEMS:
${itemList}

TOTAL: ₹${total}
    `
  };

  await transporter.sendMail(mailOptions);
  res.json({ success: true });
});

// Contact form endpoint
app.post('/api/contact', async (req, res) => {
  const { name, phone, message } = req.body;
  await transporter.sendMail({
    from: 'jayendra.jsingh@gmail.com',
    to: 'jayendra.jsingh@gmail.com',
    subject: `Website Enquiry from ${name}`,
    text: `Name: ${name}\nPhone: ${phone}\nMessage: ${message}`
  });
  res.json({ success: true });
});

app.listen(3000, () => console.log('SJ Corporation server running on port 3000'));
```

Install and run:

```bash
npm init -y
npm install express nodemailer cors
node server.js
```

Open browser → `http://localhost:3000` → your site runs with backend

# SJ Corporation — Website + Backend

## Folder Structure

```
sjcorp/
├── index.html          ← Homepage (with contact form)
├── products.html       ← Product catalogue
├── product.html        ← Single product detail
├── cart.html           ← Cart + Checkout (calls backend API)
├── admin.html          ← Admin dashboard (orders + enquiries)
├── images/             ← Put your product photos here
│   └── (your photos)
└── backend/
    ├── server.js       ← Node.js Express server
    ├── package.json    ← Dependencies
    ├── .env.example    ← Copy this to .env and fill in
    └── data/           ← Auto-created — stores orders.json + contacts.json
```

---

## STEP 1 — Install Node.js

Download and install from: https://nodejs.org  
Choose the **LTS** version. After installing, open a terminal and verify:

```bash
node --version   # should show v18 or higher
npm --version
```

---

## STEP 2 — Set up your environment file

```bash
cd sjcorp/backend
copy .env.example .env        # Windows
# OR
cp .env.example .env          # Mac / Linux
```

Open `.env` in Notepad or VS Code and fill in:

```
PORT=3000
GMAIL_USER=jayendra.jsingh@gmail.com
GMAIL_APP_PASSWORD=xxxx xxxx xxxx xxxx
NOTIFY_EMAIL=jayendra.jsingh@gmail.com
ADMIN_PASSWORD=choose_a_strong_password_here
FRONTEND_URL=*
```

### How to get your Gmail App Password:
1. Go to https://myaccount.google.com/security
2. Make sure **2-Step Verification** is ON
3. Search for **"App passwords"** at the top
4. Select App → Mail, Device → Windows/Mac → Generate
5. Copy the 16-character code into `.env` as `GMAIL_APP_PASSWORD`

---

## STEP 3 — Install dependencies and run

```bash
cd sjcorp/backend
npm install
npm start
```

You will see:

```
  ║   Admin: http://localhost:3000/admin.html ║

```

Open your browser: **http://localhost:3000**  
Your full website is now live with backend!

## API Endpoints (reference)

| Method | URL | Description |
|--------|-----|-------------|
| POST | `/api/order` | Place a new order |
| POST | `/api/contact` | Send an enquiry |
| GET | `/api/admin/orders?pass=XXX` | Get all orders |
| GET | `/api/admin/contacts?pass=XXX` | Get all enquiries |
| PATCH | `/api/admin/orders/:id?pass=XXX` | Update order status |
| DELETE | `/api/admin/orders/:id?pass=XXX` | Delete an order |
| GET | `/api/health` | Server health check |
